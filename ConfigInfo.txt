Задание 2.5 - Добавлена конфигурация для темного режима.

Поведение приложения:

При билде и запуске схемы Notes запускается приложение со светлым фоном.
При билде и запуске схемы Notes Dark запускается приложение с темным фоном.


Реализация:

1. Добавлена конфигурация DarkMode на основе Release.
Таргет Notes
    раздел Build Settings
        категория Swift Compiler - Custom Flags
            подраздел Other Swift Flags
                для конфигурации DarkMode добавлен флаг -DDARKMODE

2. Добавлена схема Notes Dark.
В редактировании схемы во вкладке Run - Info изменена Build Configuration на DarkMode.

3. В коде в файле ViewController.swift прописана логика установки фона приложения в зависимости от флага DARKMODE.
